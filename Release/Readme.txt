【DEaDlinE使用說明】

1.基本操作

  (1)主畫面 : 使用上下左右選擇按鈕，左右鍵只有在最下面要選取返回鍵的時候才需要用到，以Enter進入或退出選單。
  
  (2)角色選取 : 需要以上下左右調整，並按兩次Enter分別選取左右邊玩家的角色，或是可到最右邊一排按"下"，可以移動至返回鍵，單擊Enter返回主選單。

                *先選取的是玩家一，為左邊的玩家；後選取的是玩家二，為右邊的玩家。
		*支援兩邊相同角色。
  
  (3)地圖選取 : 進入後以上下左右選擇地圖後，單擊Enter選擇地圖，或是可到最右邊一排按"下"，可以移動至返回，單擊Enter鍵返回主選單。

  (4)教學模式 : 單擊Enter進入和退出。

  (5)遊戲設定 : 目前本區尚未開放，未來希望能由此調整遊戲平衡。

  (6)退出 : 退出遊戲，播放退出畫面。

  * 其他特殊操作 : F11可以全螢幕。

2.遊戲操作

  (1)玩家一 : W 為跳 / A為向左 / S為蹲 / D為向右 / 雙擊 S 可向下移動一層，但會判斷下方空間是否足夠決定能不能下移 / C為攻擊鍵 / V為切換武器鍵

  (2)玩家二 : I 為跳 / J為向左 / K為蹲 / L為向右 / 雙擊 K 可向下移動一層，但會判斷下方空間是否足夠決定能不能下移 / ．為攻擊鍵 / /為切換武器鍵

3.潛在問題

  *因為紅雲跳躍力極高，如果剛好撞到地圖上邊界的角落，可能會造成程式 Crash ，請使用者小心使用紅雲。
